کنترل هوشمند ویدیووال — Real-time

کنترل، چیدمان و پایش ویدیووال‌ها با یک پنل مدرن

با VideoWall Controller صحنه‌ها را طراحی کنید، منابع را روی مانیتورها بچینید و به‌صورت بلادرنگ وضعیت را پایش کنید. انیمیشن‌های اسکرولی و تعامل سه‌بعدی یک تجربه متمایز ارائه می‌دهند.

همین حالا شروع کن نمایش سه‌بعدی
Socket.IO Konva React + Vite

قابلیت‌های برجسته

چیدمان مجازی

گرید تکی/جمعی، گروه‌بندی مانیتورها و جایگذاری خودکار منابع با یک کلیک.

بلادرنگ

اتصال Socket.IO، پایش وضعیت مانیتورها و اعمال تغییرات آنی.

درگ‌ـ‌دراپ

افزودن تصویر/ویدیو/ورودی/وب به صحنه‌ها با کشیدن و رهاکردن.

مدیریت برنامه‌ها

تعریف برنامه‌ها، زمان‌بندی‌ها و سناریوهای پخش منظم.

نمایش سه‌بعدی و پارالاکس

با اسکرول، کاشی‌های ویدیووال به‌صورت استَگِر از هم جدا می‌شوند، می‌چرخند و دوباره کنار هم قرار می‌گیرند. کارت‌ها نسبت به حرکت ماوس در عمق می‌چرخند و نمایی زنده ایجاد می‌کنند.

چطور کار می‌کند؟

۱) اتصال

اتصال به درایور ویدیووال با Socket.IO (host/port از config.json).

۲) صحنه‌ها

ساخت صحنه‌ها، چیدمان مانیتورها، و افزودن منابع روی Konva Stage.

۳) اجرا

پخش برنامه‌ها، پایش وضعیت و انجام عملیات آنی.

شروع سریع

نصب وابستگی‌ها و اجرای محیط توسعه:

npm install
npm run dev
